Downloads API

La Downloads API de Python.org proporciona acceso programático a la información de versiones (releases) y archivos descargables.

Base URL

https://www.python.org/api/v2/downloads/

Endpoints

Listar releases

GET /api/v2/downloads/release/

Devuelve una lista de todas las versiones publicadas.

Detalle de un release

GET /api/v2/downloads/release/<id>/

Devuelve los metadatos de un release específico.

Archivos de un release

GET /api/v2/downloads/release_file/

Permite filtrar por `release` para obtener los archivos descargables.

Notas

  • Los resultados están paginados.
  • No se requiere autenticación.
  • Las URLs de los archivos pueden ser relativas; conviértelas a absolutas si las usas en scripts.
